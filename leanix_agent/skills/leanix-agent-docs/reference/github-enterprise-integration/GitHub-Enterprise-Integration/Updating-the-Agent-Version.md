##  Updating the Agent Version
We periodically release new versions of the [SAP LeanIX GitHub agent![Information published on non-SAP site](https://help.sap.com/doc/72d375467c1e4dcb872dfa2998b6328d/CLOUD/en-US/themes/sap-light/img/3rd_link.png)](https://help.sap.com/docs/link-disclaimer?site=https%3A%2F%2Fgithub.com%2Fleanix%2Fleanix-github-agent "https://github.com/leanix/leanix-github-agent") to ensure optimal performance. The currently deployed version is displayed at the top of the GitHub integration page. If a new version becomes available, you'll see a notification.
![Notification About a Newer Version of the GitHub Agent](https://help.sap.com/doc/72d375467c1e4dcb872dfa2998b6328d/CLOUD/en-US/loio274c0b7e7a4410148efacc61ce09e0ce_LowRes.png)
Notification About a New Version of the GitHub Agent
To update the GitHub agent to the latest version, follow these steps:
  1. Pull the latest version of the agent ghcr.io/leanix/leanix-github-agent:x.x.x.
  2. Redeploy the updated agent.
