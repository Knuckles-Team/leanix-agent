##  Save How You View Your Data in Transformations Explorer
You can save the way you are currently viewing your data, including the applied filters, column visibility, and sorting. This allows you to return to the same view at any time or switch between saved views without reapplying settings each time.
Saved views are private to the individual user who created them.
Example use cases:
  * A view called "Ready to Execute - Q1" that shows only items ready for execution in Q1, with relevant columns
  * A view called "My Team's Rollouts" that shows application rollouts filtered by your team


### Saving and Managing Views
To save your current view in the transformations explorer:
  1. Select the bookmark icon dropdown in the top right corner and select Save as New.
  2. Enter a descriptive name for the view and click Save.

From the same menu, you can load a saved view by selecting View All and choosing the view you want. You can also update, discard changes, or delete existing views.
