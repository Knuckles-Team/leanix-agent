##  Step 4: Connect SAP LeanIX to Microsoft Azure API Center
  1. In the administration area of SAP LeanIX, navigate to Integrations.
  2. Choose Add Integration.
  3. Under AI Agent Discovery, find Microsoft Azure API Center and choose Configure.
  4. Enter a name for the integration.
  5. Enter the credentials you collected in the previous step into the corresponding fields.
  6. Choose Finish.
