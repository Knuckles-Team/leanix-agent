##  Step 2: Understand the Diagram Editor
Now that you're in the editor, you'll see the following interface:
  * Canvas: Place and arrange shapes. Right‑click for context actions such as inserting from inventory, selecting by filter, or creating relations.
  * Shape library: Access categories such as General, Advanced, and Flowchart. Expand a category and drag shapes onto the canvas.
  * Toolbar and menu: Use quick actions to add shapes, insert from inventory, apply views, select by filter, or control layers.
  * Format panel: Unlink fact sheets, detach drill-downs, switch fact sheet shape templates, and adjust style preferences including colors, line types, fonts, alignment, and sizing.
