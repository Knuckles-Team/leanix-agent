##  About the Diagram Editor
The SAP LeanIX diagram editor uses draw.io technology. The interface and basic interactions, such as adding shapes, connectors, and styling, are similar to draw.io.
Key differences in LeanIX:
  * Inventory integration: Diagrams link directly to fact sheets and relations in your workspace for synchronized updates.
  * Governance: Access and editing rights follow LeanIX roles and permissions.
  * Curated feature set: Some draw.io features are limited or modified to ensure data consistency and performance.
  * Workspace storage: Diagrams are stored inside your LeanIX workspace instead of as standalone files.
