##  Step 1: Create Your First Diagram
Log in to your workspace and follow these steps:
  1. Go to the Diagrams tab and choose New Diagram.
Three diagram types are available:
     * Free Draw: Flexible canvas with general shapes and connectors. For more information, see [Free Draw Diagrams](https://help.sap.com/docs/leanix/ea/free-draw?locale=en-US&state=PRODUCTION&version=CLOUD).
     * Data Flow: Designed to model how data moves across your landscape. For more information, see [Data Flow Diagrams](https://help.sap.com/docs/leanix/ea/data-flow?locale=en-US&state=PRODUCTION&version=CLOUD).
     * Lucidchart: Connect and work with Lucidchart diagrams (create, preview, open, edit, import/export). For more information, see [Lucidchart Integration](https://help.sap.com/docs/leanix/ea/lucidchart-integration?locale=en-US&state=PRODUCTION&version=CLOUD "Seamlessly link SAP LeanIX to Lucidchart to extended visual modeling.").
  2. Select a diagram type.
  3. Enter a name for your diagram and set the permission type. ![New free draw diagram form with fields for name, permission, and description.](https://help.sap.com/doc/72d375467c1e4dcb872dfa2998b6328d/CLOUD/en-US/loio11b07507e12a462ca74dc717abb8a510_LowRes.png)
  4. Optional: To enable automatic updates, enter a description and select whether labels and view colors should update automatically based on changes in the inventory.
  5. Choose Save Diagram
The diagram editor opens.
