##  Frequently Asked Questions
Can I revert the changes from this extension?
No, this change cannot be reverted through the UI. If you need to restore the previous naming behavior, please contact support. For more, see [Contact Support](https://help.sap.com/docs/leanix/ea/contact-support?locale=en-US&state=PRODUCTION&version=CLOUD "If you encounter issues, have questions, or need assistance with SAP LeanIX, our support team is here to help. You can contact support directly within SAP LeanIX or submit a request in the SAP for Me portal.").
What happens if duplicate names are detected?
If duplicate names are detected, the extension will stop implementing the changes. You will receive a list of IT components that would result in duplicate names. Rename these IT components first, then re-run the extension.
When will the change take effect?
After validation passes, the change is scheduled to run outside of working hours. You will receive a notification once complete.
Why don’t I see the IT component naming rule extension in Optional Features and Early Access?
If you do not see this extension, your workspace was created from March 2026 onwards and already uses the updated naming convention by default.
Will this affect my existing reports or integrations?
No, this change does not affect your existing reports, dashboards, or integrations. The parent-child relationship remains unchanged, and only the display names are simplified.
