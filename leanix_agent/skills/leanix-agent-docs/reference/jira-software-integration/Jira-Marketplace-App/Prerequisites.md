##  Prerequisites
  * In SAP LeanIX:
    * Administrator access to your workspace.
    * An API token. To get an API token, create a technical user. Learn how in [Technical Users](https://help.sap.com/docs/leanix/ea/technical-users?locale=en-US&state=PRODUCTION&version=CLOUD "To get an API token, create a technical user. Manage technical users collaboratively with other administrators.").
  * Jira administrator access.


**Note**
Only Jira admins can install Atlassian apps. If you don’t have an admin account, contact your Jira admin for help with installing SAP LeanIX for Jira.
