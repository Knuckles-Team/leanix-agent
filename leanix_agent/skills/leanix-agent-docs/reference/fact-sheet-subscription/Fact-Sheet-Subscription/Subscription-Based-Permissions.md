##  Subscription-Based Permissions
Admin users can configure advanced subscription-based permissions for specific fact sheet attributes. This enables an additional validation at the fact sheet level, checking the user's subscription type before allowing or disallowing them to perform specific operations on attributes. For more information, see [Subscription-Based Permissions](https://help.sap.com/docs/leanix/ea/fact-sheet-permissions?locale=en-US&state=PRODUCTION&version=CLOUD#loio275a065b7a4410149b19cb6f3007eddd__subscription-based_permissions).
