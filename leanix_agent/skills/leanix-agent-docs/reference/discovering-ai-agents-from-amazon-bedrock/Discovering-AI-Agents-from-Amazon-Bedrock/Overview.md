##  Overview
You can automate the discovery of AI agents from Amazon Bedrock Agents and Amazon Bedrock AgentCore into SAP LeanIX. Discovered agents appear in the AI agent discovery inbox, where you can review them and decide which ones to add to your inventory or reject.
Amazon Bedrock is a platform for building generative AI applications with secure access to foundational models and supporting tools. SAP LeanIX supports dedicated integrations with two systems:
  * Amazon Bedrock Agents is a managed capability that builds and runs AI agents.
  * Amazon Bedrock AgentCore is a developer library that lets you define agents on Bedrock and run them through pluggable runtimes.


To learn more about Amazon Bedrock and how to use it, refer to the [Amazon website![Information published on non-SAP site](https://help.sap.com/doc/72d375467c1e4dcb872dfa2998b6328d/CLOUD/en-US/themes/sap-light/img/3rd_link.png)](https://help.sap.com/docs/link-disclaimer?site=https%3A%2F%2Faws.amazon.com%2Fbedrock%2F "https://aws.amazon.com/bedrock/").
**Note**
This page covers configuration steps for both Amazon Bedrock Agents and Amazon Bedrock AgentCore. The setup process is similar, but some permissions differ between the two integrations. Follow the instructions in this guide and adjust permissions based on your specific integration.
