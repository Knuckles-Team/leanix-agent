##  Inventory Filters
Filters define which fact sheets are included in the KPI calculation.
Filter mechanics:
  1. Select a fact sheet type.
  2. Add filter criteria (one or more conditions).
  3. Fact sheets must meet all criteria (AND logic).


Common filter types:
  * Lifecycle phase: Include only fact sheets in specific phases (for example, Active or End of Life).
  * Tags: Filter by assigned tags.
  * Custom fields: Use organization-specific fields.
  * Quality seals: Filter based on data quality status.
  * Relations: Include fact sheets based on connections to other objects.
  * Subscriptions: Filter by user subscriptions or ownership.
