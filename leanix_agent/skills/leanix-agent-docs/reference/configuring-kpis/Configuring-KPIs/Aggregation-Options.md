##  Aggregation Options
Operators define how fact sheet data is aggregated to produce the KPI value.
Operator | Calculation Methods | Logic | Target Field | When to Use
---|---|---|---|---
COUNT |
  * Absolute
  * Percentage

| Counts the number of fact sheets that meet filter criteria | Not applicable | Track quantities: number of fact sheets with specific attributes, tags, or quality seals
SUM |
  * Absolute
  * Percentage

| Calculates the sum of a numeric field | Numeric fields (Double or Integer type) | Calculate totals: total cost, cumulative FTE allocations, or technical debt hours
AVERAGE |
  * Absolute

| Calculates the average of a numeric field | Numeric fields (Double or Integer type) | Find mean values: average costs, mean project duration, or average complexity score
