##  Calculation Methods
KPIs support two calculation methods:
Method | How It Works | Use For | Configuration | Examples
---|---|---|---|---
Absolute | Displays the raw count or total value of the metric. |
  * Total inventory counts
  * Cumulative values
  * Raw quantities

|
  * Select COUNT, SUM, or AVERAGE
  * For SUM or AVERAGE, select the target numeric field

|
  * Total number of applications
  * Sum of all project costs
  * Count of active business capabilities
  * Completion score of fact sheets (select the AVERAGE operator)


Percentage |  Displays the metric as a percentage of a defined total. Formula: KPI = (Numerator / Denominator) x 100 |
  * Comparing subsets to the whole
  * Compliance rates
  * Proportional progress toward goals

|
  * Numerator: Define filters and aggregation for the subset you're measuring
  * Denominator: Define filters and aggregation for the total population

|
  * Percentage of cloud-native applications
  * Proportion of projects within budget
  * Rate of fact sheets with approved quality seal
