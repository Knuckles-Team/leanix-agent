The SAP LeanIX Portfolio Report groups Applications, Projects, or Providers based on specific attributes such as functional and technical suitability. This report is based on Gartner TIME assessment and can be beneficial in identifying areas that require attention. It is one of the most popular SAP LeanIX reports, supporting use cases like [Application Portfolio Assessment](https://help.sap.com/docs/leanix/ea/application-portfolio-assessment?locale=en-US&state=PRODUCTION&version=CLOUD "Establish transparency and governance over your as-is landscape to identify optimization opportunities and unlock further use cases.") and [Application Rationalization](https://help.sap.com/docs/leanix/ea/application-rationalization?locale=en-US&state=PRODUCTION&version=CLOUD "Application rationalization reduces costs, eliminates redundancies, and improves efficiency. Optimize your application landscape with this step-by-step approach.").
The Portfolio Report offers a high-level summary or big picture of an organization’s portfolio state to support decision-making in discussions between the EA and stakeholders from the business.
