##  Detailed Guides
SAP Cloud ALM Integration involves these aspects:
  * Enabling API access to SAP Cloud ALM and connecting SAP LeanIX to the SAP Cloud ALM API. For a detailed guide, see [Configuring SAP Cloud ALM Integration for Managing Projects](https://help.sap.com/docs/leanix/ea/configuring-sap-cloud-alm-integration-for-managing-projects?locale=en-US&state=PRODUCTION&version=CLOUD "Integrate SAP Cloud ALM with SAP LeanIX by setting up an SAP Cloud ALM instance, enabling API access, and connecting SAP LeanIX workspace to SAP Cloud ALM API.").
  * Reviewing and importing the projects from SAP Cloud ALM to SAP LeanIX and exporting initiatives from SAP LeanIX to Cloud ALM. For a detailed guide, see [Synchronizing Projects Between SAP Cloud ALM and SAP LeanIX](https://help.sap.com/docs/leanix/ea/synchronizing-projects-between-sap-cloud-alm-and-sap-leanix?locale=en-US&state=PRODUCTION&version=CLOUD "Learn how to synchronize projects between SAP Cloud ALM and SAP LeanIX, including linking existing fact sheets, importing projects as new fact sheets, exporting initiatives to SAP Cloud ALM, running manual syncs, and delinking fact sheets.").
