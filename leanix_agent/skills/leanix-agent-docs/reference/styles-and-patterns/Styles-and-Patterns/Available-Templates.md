##  Available Templates
  * SAP LeanIX Default: Adheres to SAP LeanIX's visual standards. It provides a consistent look and feel for fact sheet shapes and colors. This template is ideal if you want to use the default settings provided by SAP LeanIX.
  * Archimate 3.2: Follows the Archimate 3.2 visual standards. It is useful if you prefer a more structured approach and are familiar with Archimate modeling.
  * Custom Template: Tailored to your organization's specific requirements. Customize shapes, colors, and configurations to align with your unique brand identity and data visualization needs.
  * SAP Business Technology Platform (BTP) Reference Architecture Patterns: Based on SAP BTP reference architecture patterns. These templates are particularly useful for organizations using SAP BTP, providing a framework that aligns with SAP's architecture best practices.
