##  Procedure
  1. From the [SAP for Me](https://help.sap.com/docs/link-disclaimer?site=https%3A%2F%2Fme.sap.com "https://help.sap.com/docs/link-disclaimer?site=https%3A%2F%2Fme.sap.com") home page, choose Portfolio & Products.
  2. On the My Product Packages tab, select the tile SAP LeanIX.
  3. On the Applications tab, choose Start Provisioning.
The Create Solution details window opens.
  4. Enter the necessary information and continue with Next.
  5. Select applications and continue with Next.
All applications you ordered are preselected.
  6. Continue with Review.
All settings are shown on one page.
  7. Confirm with Finish to start the provisioning.
In the Resources tab, you can view the status.
  8. The IT main admin receives an e-mail notification with the necessary URLs and information related to integration. Complete the steps described in the e-mail.
  9. Your workspace name, URL, and tenant ID is automatically determined based your data in the SAP CRM system. If you require a different workspace name, reach out to the support team.


If you encounter any issues during provisioning, contact support. Learn how in [Contact Support](https://help.sap.com/docs/leanix/ea/contact-support?locale=en-US&state=PRODUCTION&version=CLOUD "If you encounter issues, have questions, or need assistance with SAP LeanIX, our support team is here to help. You can contact support directly within SAP LeanIX or submit a request in the SAP for Me portal.").
