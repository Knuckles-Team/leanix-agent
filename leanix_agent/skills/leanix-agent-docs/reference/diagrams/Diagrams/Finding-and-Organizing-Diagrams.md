##  Finding and Organizing Diagrams
Search, filter, and sort functions help you streamline navigation, save time, and maintain better control over large sets of diagrams.
  * Search and filter by ownership, read and write permissions, creation date, modification date, and diagram type.
  * Sort diagrams by attributes such as name, date, or relevance to find what you need faster.
  * Switch between grid view (visual browsing) and list view (tabular details) depending on how you want to work with diagrams.
