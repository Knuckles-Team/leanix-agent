##  Submitting a Request in SAP LeanIX
### Context
Learn how to access support directly from the main navigation in SAP LeanIX. For more information about using built-in support, refer to [Built-In Support](https://help.sap.com/docs/built-in-support "https://help.sap.com/docs/built-in-support").
### Procedure
  1. To open the built-in support panel, choose the headset icon in the main navigation.
The panel opens with a search bar and a list of recommended links. If your SAP Universal ID is linked to an S-user account, the built-in support panel shows a summary of your support cases.
  2. Choose Report New Issue to create a new support request.
