##  Best Practices for Managing Your Inbox
Follow these recommendations to manage your SAP discovery inbox:
  * Choose the integration type you want to work with. Filter your inbox to show only those items.
  * Review all action items. Make sure each Action Needed  entry is linked or rejected.
  * Set a monthly reminder to check new inbox items.
  * Track system status with lifecycle indicators such as in plan, phase in, or active.
  * Assign a responsible person for each fact sheet you create.
