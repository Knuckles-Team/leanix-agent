##  Overview
This guide explains how to integrate SAP LeanIX with SAP Cloud ALM to automate the discovery of SAP systems, services, and integration flows. The setup process includes these key steps:
  1. Provision your Cloud ALM instance.
  2. Generate a service key in SAP BTP.
  3. Use the service key to establish a connection with LeanIX.
  4. Optionally, enable additional capabilities to extend the integration, which allows you to:
     * Capture SAP system-level information
     * Discover integration flows between systems
For details, refer to [Additional Capabilities](https://help.sap.com/docs/leanix/ea/sap-discovery-configuring-cloud-alm?locale=en-US&state=PRODUCTION&version=CLOUD#loio2759721b7a441014bd508464cc06d9fa__Additional-Capabilities).


After you complete the setup, link discovered systems to fact sheets to enhance visibility and support effective modeling.
