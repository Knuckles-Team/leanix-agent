##  How KPIs Are Calculated
KPIs are calculated in two phases:
Initial calculation:
  * Occurs immediately when you create or edit a KPI
  * Duration ranges from a few seconds to a few minutes
  * Depends on the complexity of filters and the dataset size


Ongoing recalculation:
  * Occurs when inventory data involved in the KPI calculation changes
  * Tracks trends and maintains historical values for analysis
  * Allows you to monitor changes over time


**Note**
Historical values are not recalculated when you modify the calculation rule for a KPI. If you change the calculation rule, historical values reflect the old formula. For significant changes, consider creating a new KPI instead.
