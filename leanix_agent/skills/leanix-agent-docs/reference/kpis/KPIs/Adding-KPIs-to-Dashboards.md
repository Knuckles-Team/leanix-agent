##  Adding KPIs to Dashboards
You can display KPIs on dashboards using the KPI panel. To add a KPI to a dashboard, follow these steps:
  1. Open a dashboard you can edit or create a new dashboard.
  2. On the right-side panel, select KPI Panel and drag it to the desired position on the dashboard.
![Screenshot showing the dashboard editor with the KPI Panel option in the right-side panel ready to be dragged onto the dashboard.](https://help.sap.com/doc/72d375467c1e4dcb872dfa2998b6328d/CLOUD/en-US/loio1caa87689bc041628e8585a91d6bfafb_LowRes.png)
KPI Panel Selection
  3. In the panel configuration dialog, specify the following:
     * Panel Title: A descriptive title for the panel.
     * Main KPI: The primary KPI to display, including a historical trend chart.
     * Additional KPIs: Optional additional KPIs to display below the main KPI.
  4. Choose Save.
  5. Add more panels to the dashboard as needed.
  6. To save changes to the dashboard, choose Save.


The KPI panel now displays your selected KPIs with their current values and trends.
