##  Permissions
By default, only admins can manage collections and their content. Admins can grant users in other roles the permissions to:
  * Administrate: Create and manage collection groups and collections.
  * Manage content: Add and remove dashboards, reports, and diagrams within collections.


To modify collection permissions for non-admin roles, navigate to the User Roles and Permissions section of the admin area. For details, see [Role-Based Permissions](https://help.sap.com/docs/leanix/ea/user-roles-and-permissions?locale=en-US&state=PRODUCTION&version=CLOUD#loio275def827a4410149362f3672c614956__role-based_permissions).
