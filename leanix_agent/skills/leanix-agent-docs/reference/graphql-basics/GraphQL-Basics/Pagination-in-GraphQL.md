##  Pagination in GraphQL
The SAP LeanIX GraphQL API employs the concept of [relay cursor connections![Information published on non-SAP site](https://help.sap.com/doc/72d375467c1e4dcb872dfa2998b6328d/CLOUD/en-US/themes/sap-light/img/3rd_link.png)](https://help.sap.com/docs/link-disclaimer?site=https%3A%2F%2Frelay.dev%2Fgraphql%2Fconnections.htm "https://relay.dev/graphql/connections.htm") for pagination, allowing for efficient and flexible data retrieval. In this pagination model, instead of using offsets, the API uses cursors to navigate through the data set. For more details, see [Pagination in GraphQL](https://help.sap.com/docs/leanix/ea/pagination-in-graphql?locale=en-US&state=PRODUCTION&version=CLOUD "Learn how pagination in GraphQL works and how to paginate results in your queries.").
