##  Logical Operators
Logical operators are essential tools for filtering complex data sets in GraphQL. They let you combine or modify filter conditions, enabling you to create more complex and precise data queries.
The table below provides an overview of the logical operators you can use in your GraphQL queries:
Logical Operator | Description
---|---
AND | Returns true if all specified conditions are met. It allows you to narrow down results by requiring multiple criteria to be satisfied.
OR | Returns true if at least one specified condition is met. It allows you to broaden results by including data that satisfies any of the given criteria.
NOR | Returns true if none of the specified conditions are met. It allows you to exclude data that satisfies any of the given criteria.
