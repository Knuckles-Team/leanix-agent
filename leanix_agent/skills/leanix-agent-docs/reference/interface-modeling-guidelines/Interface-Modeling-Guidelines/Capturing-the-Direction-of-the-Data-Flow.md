##  Capturing the Direction of the Data Flow
To record the direction of the data flow itself, use the dedicated field in the Data & Technology section of the interface fact sheet. It can be one of the following:
  * Incoming: Data flow is from consumer applications to the interface provider application.
  * Outgoing: Data flow is from the interface provider application to consumer applications.
  * Bi-Directional: Data flow in both directions between interface provider application and consumer applications.
