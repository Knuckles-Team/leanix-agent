##  API Endpoints Used by the Integration
The integration uses Jira REST API endpoints for project discovery, field mapping, and issue synchronization in both cloud and on-premise setups.
Jira Cloud uses Atlassian /rest/api/3/ endpoints. Refer to the [Jira Cloud REST API![Information published on non-SAP site](https://help.sap.com/doc/72d375467c1e4dcb872dfa2998b6328d/CLOUD/en-US/themes/sap-light/img/3rd_link.png)](https://help.sap.com/docs/link-disclaimer?site=https%3A%2F%2Fdeveloper.atlassian.com%2Fcloud%2Fjira%2Fplatform%2Frest%2Fv3%2Fintro%2F "https://developer.atlassian.com/cloud/jira/platform/rest/v3/intro/") for more information.
If your on-premise Jira instance is behind a reverse proxy or API gateway, you may need to verify endpoints. You can find the full list of on-premise endpoints in [Step 1: Add the Jira Integration](https://help.sap.com/docs/leanix/ea/setting-up-jira-cloud-projects-integration?locale=en-US&state=PRODUCTION&version=CLOUD#loiobfb8ad2239684b649229834b793f09dc__Step-1_-Add-the-Jira-Integration).
