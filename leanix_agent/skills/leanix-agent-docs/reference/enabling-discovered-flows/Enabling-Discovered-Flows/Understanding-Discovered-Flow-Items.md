##  Understanding Discovered Flow Items
Each discovered flow item in your inbox contains the following information:
  * Integration type: The category of integration (for example, IDoc messages or RFC calls)
  * Involved systems: The SAP systems that participate in the flow
  * Discovery metadata, including:
    * Discovery date: When the system first discovered the flow
    * Last seen: When the system last observed the flow
    * Number of involved systems: The total count of systems in the flow
