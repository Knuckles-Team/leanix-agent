##  Troubleshooting
No discovered flows appear in the inbox
  * Verify that Integration & Exception Monitoring is enabled in SAP Cloud ALM.
  * Verify that the monitored systems have active integration traffic.
  * Verify that data collection is active for each system you want to discover.
  * Verify that flow discovery is enabled in SAP Cloud ALM.
  * Review the sync log for errors related to Cloud ALM Flow API.


Flows are missing or incomplete
  * Verify that all relevant systems are onboarded to SAP Cloud ALM monitoring.
  * Check the network connectivity between the systems and Cloud ALM.
  * Verify that the integration scenarios you expect are active and monitored.


Service key authentication errors
  * Verify that the service key includes the required scopes.
  * Test the connection in the SAP LeanIX integration settings.
  * Regenerate the service key if needed.
