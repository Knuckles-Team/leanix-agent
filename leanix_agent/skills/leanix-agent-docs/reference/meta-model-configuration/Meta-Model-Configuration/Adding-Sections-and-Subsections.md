##  Adding Sections and Subsections
Adding custom sections and subsections to a fact sheet provides structure for new fields and enables you to arrange data in a specific order, enhancing readability and organization.
Here’s how you can add these elements:
  * To add a custom section, select an existing section, click Add section, then configure section parameters on the right-side panel.
  * To add a custom subsection, select an existing section or subsection, click Add subsection, then configure subsection parameters on the right-side panel.
