##  Creating a Diagram Template
Every diagram template is based on a single base fact sheet type, for example, application or business capability. When you create a template, you can start from scratch or use a preset that comes with predefined relations and drill-downs for common use cases.
  1. On the Diagram Templates page, choose New Template.
  2. Select a preset or start with a blank template.
  3. Enter a name for the template and an optional description.
  4. If starting from a blank template, set a base fact sheet type.
  5. Choose Save Template.


The template editor opens with your configured base fact sheet. Explore the types of elements you can add to your template in [Template Elements at a Glance](https://help.sap.com/docs/leanix/ea/creating-and-editing-diagram-templates?locale=en-US&state=PRODUCTION&version=CLOUD#loiodc871529cf1046a4abcf4f183c9b6985__Template-Elements-at-a-Glance).
