##  Adding Dependencies to a Template
Dependency relations show peer-level connections between fact sheets without visual hierarchy. You can still use dependencies to represent hierarchical structures.
  1. Right-click the source fact sheet type.
  2. In the menu, choose Show Dependency and select the dependency type.
For example, “Business Capabilities.”
