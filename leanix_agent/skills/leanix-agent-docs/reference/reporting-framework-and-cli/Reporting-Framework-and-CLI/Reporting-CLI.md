##  Reporting CLI
The [Reporting CLI![Information published on non-SAP site](https://help.sap.com/doc/72d375467c1e4dcb872dfa2998b6328d/CLOUD/en-US/themes/sap-light/img/3rd_link.png)](https://help.sap.com/docs/link-disclaimer?site=https%3A%2F%2Fgithub.com%2Fleanix%2Fleanix-reporting-cli "https://github.com/leanix/leanix-reporting-cli") allows you to initialize, develop, and publish custom reports.
To learn how to use the CLI tool, run the lxr --help command. To get detailed help for individual commands and their options, run lxr help [command].
