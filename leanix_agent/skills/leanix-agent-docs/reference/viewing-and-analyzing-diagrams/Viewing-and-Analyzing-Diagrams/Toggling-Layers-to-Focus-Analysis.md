##  Toggling Layers to Focus Analysis
Layers help you visualize different levels of abstraction in architecture diagrams. You can show or hide layers when viewing and editing diagrams to focus on specific architectural elements.
When viewing a diagram, toggling layers allows you to:
  * Analyze specific relations or components in isolation
  * Understand how elements integrate into the broader architecture
  * Create focused views for different audiences


To toggle layer visibility:
  1. Hover over your diagram to display the toolbar.
  2. Choose Layers.
![Diagram editor with the Layers button highlighted in the toolbar.](https://help.sap.com/doc/72d375467c1e4dcb872dfa2998b6328d/CLOUD/en-US/loio29c5113521ad4cc4916d1468957e3f7b_LowRes.png)
  3. Choose the eye icon next to a layer name to show or hide it.


**Note**
You can toggle layer visibility without edit permissions. To add new layers, you need edit permissions. For more information about adding layers to diagrams, see [Adding Layers](https://help.sap.com/docs/leanix/ea/free-draw?locale=en-US&state=PRODUCTION&version=CLOUD#loio275a369c7a4410148ff4e12bcf658e03__adding_layers).
