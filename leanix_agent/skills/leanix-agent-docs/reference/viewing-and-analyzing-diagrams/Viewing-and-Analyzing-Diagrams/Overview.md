##  Overview
Diagrams in SAP LeanIX help you visualize your enterprise architecture. When you view a diagram, you can apply views and filters, examine fact sheet details, and analyze relations without changing the diagram structure.
This guide covers viewing capabilities that work with both data flow and free draw diagrams. These features help you:
  * Analyze architecture and dependencies visually
  * Filter and focus on specific elements
  * Examine fact sheet details without leaving the diagram
  * Color code elements by attributes such as lifecycle or business criticality
