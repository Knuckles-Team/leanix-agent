##  Viewing Linked Architecture Decisions
When you view a diagram, you can see which architecture decisions are linked to it. These links make it easy to understand the context behind the diagram and trace which decisions impact your architecture.
**Restriction**
This feature is available only if you use SAP LeanIX Architecture and Road Map Planning.
To view linked architecture decisions:
  1. Open a diagram.
  2. Choose Architecture Decisions.
  3. In the dialog box, review the list of architecture decisions linked to the diagram.
![Dialog showing a list of linked architecture decisions with ID, title, status, creator, and date.](https://help.sap.com/doc/72d375467c1e4dcb872dfa2998b6328d/CLOUD/en-US/loiof92c5a3f04d340b38546ff298d5df041_LowRes.png)
  4. Select a decision to open it.
If no architecture decisions reference the diagram, the dialog box provides a link to the list of architecture decisions. You can create a new decision or view an existing one from there.


For more information about architecture decisions, see [Architecture Decisions](https://help.sap.com/docs/leanix/ea/architecture-decisions?locale=en-US&state=PRODUCTION&version=CLOUD "Document and manage enterprise architecture decisions directly within SAP LeanIX, keeping all architecture-related information in one place.").
**Note**
Architecture decisions link to the current diagram version only. Previous diagram versions are not linked.
