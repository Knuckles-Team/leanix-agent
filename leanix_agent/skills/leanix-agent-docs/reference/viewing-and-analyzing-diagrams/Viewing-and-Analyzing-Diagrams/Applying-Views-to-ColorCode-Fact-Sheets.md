##  Applying Views to Color‑Code Fact Sheets
Diagram views let you visualize data from fact sheets with color‑coded attributes and relations. These views are consistent with report views, providing a unified experience across diagrams and reports.
The views available in diagrams match the fact sheet fields and relations defined in the SAP LeanIX meta model. The available views depend on the fields configured in your workspace. Any custom fields that you add to fact sheets also appear in the views list.
You can apply views when viewing and editing diagrams. This section focuses on using views to analyze and explore diagrams. To learn how to use views while designing diagrams, see [Applying and Aggregating Views in Free Draw](https://help.sap.com/docs/leanix/ea/free-draw?locale=en-US&state=PRODUCTION&version=CLOUD#loio275a369c7a4410148ff4e12bcf658e03__applying_and_aggregating_views) and [Applying and Aggregating Views in Data Flow](https://help.sap.com/docs/leanix/ea/data-flow?locale=en-US&state=PRODUCTION&version=CLOUD#loio2759ae097a441014acff8ab536c79254__applying_and_aggregating_views).
To apply a view:
  1. Choose Select a view.
  2. In the view menu, select your fact sheet type and preferred view.


After you select a view, the system applies the view colors to the related fact sheets and displays a legend at the top of the page. The view remains when you export the diagram or add it to a dashboard.

**Note**
When you open a diagram, an Updating diagram spinner icon may appear. This means the system is loading data for views and updating the diagram in the background.
While the spinner is visible, you can do the following:
  * Zoom in and out
  * Change the viewport
  * Select fact sheets and relations
  * Apply filters


When the update finishes, the spinner disappears and the toolbar becomes available. You can then use all toolbar actions.
