Applied filters are preserved when you:
  * Export diagrams: Filters remain in PDF, SVG, PNG, and other export formats, providing a consistent viewing experience for recipients.
  * Add diagrams to dashboards: Filters persist in dashboard widgets, ensuring the same filtered view appears for all viewers.


If you switch between viewing and editing, the system resets any unsaved filter changes to their previous state. To keep your filter changes, save the diagram.
