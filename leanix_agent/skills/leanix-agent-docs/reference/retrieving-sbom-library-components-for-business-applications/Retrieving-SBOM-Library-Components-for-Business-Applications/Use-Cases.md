##  Use Cases
  * License compliance: Query components for a business application and flag restricted or unknown licenses before deployment. You can automate this process by integrating the endpoint with your CI/CD pipeline. You can also build custom dashboards for component analysis across your application portfolio.
  * Application risk assessment: Retrieve the full component inventory to assess exposure to known vulnerabilities at the application level. For example, you can identify applications using vulnerable libraries and prioritize remediation efforts.
  * AI-powered governance: Connect AI agents through MCP to answer questions such as "Which libraries does this application use?" or automate component audits across applications.
