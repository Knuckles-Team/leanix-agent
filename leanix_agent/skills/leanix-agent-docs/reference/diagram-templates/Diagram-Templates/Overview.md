##  Overview
Diagram templates define reusable rules for how your data is visualized. You can view your landscape in standardized diagrams that always reflect the current inventory without redesigning layouts. These templates provide an instant, dynamic preview of your landscape for exploration and analysis. You can also use them to create fixed diagrams for documentation, sharing, and audits.
Typically, administrators create and publish templates. Users can then explore the dynamic preview and save a snapshot as a static diagram for documentation or sign-off.
