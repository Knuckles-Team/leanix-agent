##  Diagram Templates versus Standard Diagrams
Tool | Diagram Templates (Dynamic Diagrams) | Free Draw and Data Flow (Standard Diagrams)
---|---|---
Purpose | Real-time exploration of your landscape with standardized, reusable views. | Documentation, presentation, or custom visualizations with full editorial control.
Data source | Always live. | Captured at a point in time, but can be refreshed manually.
When to use |
  * Daily architecture analysis
  * Exploring current dependencies
  * Sharing up-to-date views with stakeholders
  * Quick landscape checks

|
  * Documentation that won't change significantly
  * Presentations and reports
  * Diagrams needing custom annotations or shapes


Typical workflow |
  * Admins create and publish templates
  * Admins and users explore the template preview as a dynamic diagram for analysis

|
  * Admins and users create a static diagram from scratch or from a preview when a fixed, editable snapshot is needed, for example, for documentation or audits.
