##  Key Features
  * Template‑based diagram creation: Define reusable templates that determine the structure and content of diagrams created from inventory data.
  * Inventory‑synchronized rendering: Create diagrams that reflect the current state of your inventory.
  * Interactive previews and saved snapshots: Inspect live inventory context, for example, which applications are affected by an initiative, or save rendered diagrams to capture a point in time.
  * Predefined views and labels: Use views and labels in the template to control color-coding and highlight key information.
