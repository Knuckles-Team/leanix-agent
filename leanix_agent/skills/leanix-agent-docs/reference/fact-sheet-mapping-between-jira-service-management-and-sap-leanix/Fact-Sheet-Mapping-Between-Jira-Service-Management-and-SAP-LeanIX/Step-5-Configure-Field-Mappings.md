##  Step 5: Configure Field Mappings
Define which attributes to synchronize between the systems:
  1. Review the available fields from both systems displayed.
  2. For each mapping:
     * SAP LeanIX Field: Select the field from SAP LeanIX.
     * Jira Service Management Field: Select the corresponding field type in Jira Service Management.
     * The mapping direction follows the source system defined in Step 2.
  3. Add additional field mappings as needed using Add Field Mapping.
  4. Choose Save to complete the mapping configuration.


### Field Mapping Parameters
Parameter | Description
---|---
LeanIX Field | Field key in SAP LeanIX.
Direction | Synchronization direction determined by the source system.
Jira Field | Field type in Jira Service Management.
