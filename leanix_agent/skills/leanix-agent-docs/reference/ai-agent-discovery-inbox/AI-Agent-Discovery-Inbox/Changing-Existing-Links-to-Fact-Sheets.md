##  Changing Existing Links to Fact Sheets
You can change the link between a discovered item and a fact sheet if a link was added by mistake or if a more suitable option is available.
To change a link, do the following:
  1. Choose the name of a discovered item to open the side panel.
  2. Hover over the linked fact sheet and choose Edit.
  3. Search for and select the desired fact sheet.
![](https://help.sap.com/doc/72d375467c1e4dcb872dfa2998b6328d/CLOUD/en-US/loio7564cf0ed4fd46b9a4edf81a45d20b14_LowRes.gif)
Changing a Linked Fact Sheet
  4. Choose Add to Inventory.


**Tip**
To reject a previously linked item without making any changes, choose Reject.
