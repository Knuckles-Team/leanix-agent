##  Step 2: Create an SAP Cloud Management Service Instance and Service Key
To create a service instance and key, follow the instructions in [_Getting an Access Token for SAP Cloud Management Service APIs_](https://help.sap.com/docs/btp/sap-business-technology-platform/getting-access-token-for-sap-cloud-management-service-apis?locale=en-US "https://help.sap.com/docs/btp/sap-business-technology-platform/getting-access-token-for-sap-cloud-management-service-apis?locale=en-US").
Use the "central-viewer" service plan (recommended) or higher for your SAP Cloud Management service instance.
