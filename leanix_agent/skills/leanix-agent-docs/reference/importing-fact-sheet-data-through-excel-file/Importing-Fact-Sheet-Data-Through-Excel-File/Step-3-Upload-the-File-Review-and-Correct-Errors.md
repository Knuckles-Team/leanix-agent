##  Step 3: Upload the File, Review, and Correct Errors
To upload the file:
  1. Go to the Inventory.
  2. From the Inventory Tools drop-down menu, choose Import from Excel.
![Importing from Excel](https://help.sap.com/doc/72d375467c1e4dcb872dfa2998b6328d/CLOUD/en-US/loiob601b46d6dcb43c9b47fde2cc416f079_LowRes.png)
  3. Drag and drop the file into the import overlay, or click Browse to select a file. SAP LeanIX will check the file and generate a preview.


During the upload process, you can minimize the import window by clicking Continue in Background. This allows you to continue working in other areas while the file is being processed. A progress widget in the inventory sidebar lets you monitor progress and reopen the import window to view more details at any time.
**Tip**
  * Closing your browser or refreshing the page doesn't stop the import process. When you log back in, the active import is automatically detected, and the progress widget appears again in the inventory sidebar.
  * Only one import can run at a time. A notification prevents you from starting another import while one is already in progress.


The uploaded file undergoes a check before it is actually imported into your workspace. This check allows you to review and confirm the changes before proceeding with the import. You will receive a summary of the proposed changes. Choose Import to complete the upload when you are satisfied.
![Previewing Import](https://help.sap.com/doc/72d375467c1e4dcb872dfa2998b6328d/CLOUD/en-US/loio0e8e0c04d32e4205b5c3cb44d527782b_LowRes.png)
Preview of Import Overlay
An overview of errors is also provided if there are any. You need to resolve these errors before you can import the file. To review and identify the errors, download the summary report. The summary report details the changes being made to the fact sheets and the details of the errors row by row. You can fix them and re-upload the same file.
