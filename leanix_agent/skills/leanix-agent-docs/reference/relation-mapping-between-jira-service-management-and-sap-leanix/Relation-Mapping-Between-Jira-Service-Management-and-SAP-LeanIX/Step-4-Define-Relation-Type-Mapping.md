##  Step 4: Define Relation Type Mapping
Map the relation types between systems:
  1. SAP LeanIX Relation Type: Select the relation type from SAP LeanIX. The dropdown displays available relation types based on the selected fact sheet types.
  2. Jira Service Management Relation Type: Select the corresponding relation type from Jira Service Management. The dropdown displays available reference types from your Jira Service Management schema.
  3. Choose Save to complete the relation mapping configuration.
