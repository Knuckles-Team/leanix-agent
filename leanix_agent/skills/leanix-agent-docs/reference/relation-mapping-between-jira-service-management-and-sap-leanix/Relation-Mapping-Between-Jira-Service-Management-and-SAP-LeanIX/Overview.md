##  Overview
Relation mapping is optional and defines which fact sheet relations are synchronized between SAP LeanIX and Jira Service Management. Configuring relation mappings enables automatic documentation of dependencies identified in one system within the other system.
Each relation mapping configuration specifies:
  * Source system: Which system serves as the source of truth for the relation.
  * Fact sheet mappings: Which previously configured fact sheet mappings define the relation endpoints.
  * Relation types: Which relation types correspond between the systems.
