##  Searching for Filter Groups
With numerous filter criteria available, being able to search for specific filter groups enables you to quickly find the most relevant options for your needs. For example, if you are looking for filters related to application lifecycle status, you can search for ‘lifecycle’ and immediately see all lifecycle filter options.
