##  Self-Built Software Agent
The [SAP LeanIX Self-Built Software Agent![Information published on non-SAP site](https://help.sap.com/doc/72d375467c1e4dcb872dfa2998b6328d/CLOUD/en-US/themes/sap-light/img/3rd_link.png)](https://help.sap.com/docs/link-disclaimer?site=https%3A%2F%2Fgithub.com%2FSAP%2Fleanix-self-built-software-agent%2F "https://github.com/SAP/leanix-self-built-software-agent/") is an open source CLI tool and library designed to automatically discover and analyze self-built software within your GitHub organization. Using AI agents powered by LangChain and LangGraph, it identifies services, dependencies, technology stacks, and runtime details without requiring manual intervention. This automated approach helps organizations maintain accurate software catalogs, manage technical debt, and gain comprehensive visibility into their software portfolio.
The tool supports multiple LLM providers, including OpenAI, Azure OpenAI, and Anthropic Claude, giving you flexibility in choosing your preferred AI backend. The sophisticated multi-agent workflow, built on LangGraph, enables intelligent repository classification and service discovery that adapts to your organization's specific patterns and naming conventions.
