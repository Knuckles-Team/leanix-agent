##  Deleting Import Rulesţéł
To permanently remove an import rule, do the following:
  1. In the administration area of SAP LeanIX, go to Integrations > Atlassian Jira and select the integration instance.
  2. On the Import Rules tab, choose the bin icon next to the rule you want to delete.
  3. In the confirmation message, choose Delete.
