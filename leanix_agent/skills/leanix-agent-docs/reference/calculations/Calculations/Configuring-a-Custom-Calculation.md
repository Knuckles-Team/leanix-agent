##  Configuring a Custom Calculation
Follow these steps:
  1. In the administration area, go to the Calculations section.
  2. Click New Calculation.
  3. In the Other section, on the Create from Scratch tile, click Create. Alternatively, you can create a calculation from a template.
![Creating a Custom Calculation from Scratch](https://help.sap.com/doc/72d375467c1e4dcb872dfa2998b6328d/CLOUD/en-US/loio274e6e657a4410149319b3d021bdd60c_LowRes.png)
Creating a Custom Calculation from Scratch
  4. Configure the calculation:
    1. Enter a name and, optionally, a description for the calculation.
    2. Select the owner of the calculation template.
    3. Select a target fact sheet type and attribute.
    4. Enter JavaScript code to define the calculation logic. For more information, see [Technical Details](https://help.sap.com/docs/leanix/ea/calculations?locale=en-US&state=PRODUCTION&version=CLOUD#loio275919c67a441014a04fa34f480310eb__technical_details).
  5. Test the calculation to ensure it works as expected. For details, see [Testing a Calculation](https://help.sap.com/docs/leanix/ea/calculations?locale=en-US&state=PRODUCTION&version=CLOUD#loio275919c67a441014a04fa34f480310eb__testing_a_calculation).
  6. Save the calculation as a draft or activate it immediately after saving.
