##  Linking / Rejecting Multiple Items at Once
To link or reject multiple items at once, follow these steps:
  1. From the list, select the discovered items by checking the box next to the items. Checking the checkbox in the title row selects all currently displayed items. To select the entire list, scroll to the bottom of the page to load all items before selecting the title row checkbox. You can also use filters (e.g., by integration source) to narrow down the list and only select those.
  2. At the top right corner of the list, click Link or Reject as needed.
