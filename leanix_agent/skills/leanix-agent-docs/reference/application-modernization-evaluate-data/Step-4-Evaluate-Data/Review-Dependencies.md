##  Review Dependencies
Start your evaluation by identifying any dependencies that might impact your modernization. This ensures that your transformation plan is both technically sound and strategically aligned. For example, if an application needs to be replatformed and shares a database with another application, it may be more efficient to migrate both together to retire the shared IT component.
You can do this with free draw diagrams and interface circle maps.
