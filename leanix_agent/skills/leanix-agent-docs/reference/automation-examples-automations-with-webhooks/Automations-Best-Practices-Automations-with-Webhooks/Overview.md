##  Overview
You can set up automations that send a webhook to a target URL. The following examples show basic scenarios you can use as a starting point. Each use case includes an action to configure in your automation platform. For detailed instructions, refer to the documentation for your automation platform.
To learn more about automations with webhooks, see:

  * [Creating an Automation with a Webhook Action](https://help.sap.com/docs/leanix/ea/automations-webhooks?locale=en-US&state=PRODUCTION&version=CLOUD "Send webhooks from automations to trigger external workflows.")
