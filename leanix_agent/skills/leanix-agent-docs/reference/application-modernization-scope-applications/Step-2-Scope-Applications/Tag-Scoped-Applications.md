##  Tag Scoped Applications
We recommend tagging scoped applications for easy identification in the inventory, reports, surveys, and dashboards. You can create individual tags or cluster a series of related tags within a tag group.
**Example**
If you’re assessing applications for a cloud transformation, for example, you might create a tag group called “Cloud Transformation”. Within that group, you create the tags “Private Cloud,” “Public Cloud,” and “Hybrid,” to indicate the targeted type of cloud environment for each application.
