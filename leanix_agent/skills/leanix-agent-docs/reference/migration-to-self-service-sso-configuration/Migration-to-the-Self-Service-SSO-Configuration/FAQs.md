### We already have an existing IdP app. Can we clone it?
Most identity providers don't allow cloning applications. Start from scratch with a new application in your IdP.
### Where do I find Entity ID and Reply URL?
  1. In SAP LeanIX, go to Administration > Authentication and SSO.
  2. When you start adding a new IdP, the Entity ID and Reply URL are generated and displayed.
