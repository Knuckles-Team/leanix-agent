##  Create an API Token in Zscaler
**Note**
  * Each organization can only have one API key. For more details, see [Zscaler Cloud Service API Key![Information published on non-SAP site](https://help.sap.com/doc/72d375467c1e4dcb872dfa2998b6328d/CLOUD/en-US/themes/sap-light/img/3rd_link.png)](https://help.sap.com/docs/link-disclaimer?site=https%3A%2F%2Fhelp.zscaler.com%2Fzia%2Fabout-cloud-service-api-key "https://help.zscaler.com/zia/about-cloud-service-api-key").
  * Ensure that the language for the integration user in Zscaler is set to English.


  1. Log out and log in again using the newly created user account.
  2. Navigate to Administration → Cloud Service API Security.
  3. In the Cloud Service API Key tab, select + Add API Key to create a new key.
