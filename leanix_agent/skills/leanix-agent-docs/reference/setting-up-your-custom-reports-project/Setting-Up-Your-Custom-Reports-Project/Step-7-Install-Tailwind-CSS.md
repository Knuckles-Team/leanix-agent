##  Step 7: Install Tailwind CSS
To be able to style your custom reports, install a CSS framework.
**Note**
You can use your preferred CSS library to build custom reports. In this tutorial, we use Tailwind CSS. For complete setup instructions, refer to the official [Tailwind CSS documentation![Information published on non-SAP site](https://help.sap.com/doc/72d375467c1e4dcb872dfa2998b6328d/CLOUD/en-US/themes/sap-light/img/3rd_link.png)](https://help.sap.com/docs/link-disclaimer?site=https%3A%2F%2Ftailwindcss.com%2Fdocs%2Finstallation "https://tailwindcss.com/docs/installation").
To install Tailwind CSS, run the following command:

```
npm install -D tailwindcss
```
