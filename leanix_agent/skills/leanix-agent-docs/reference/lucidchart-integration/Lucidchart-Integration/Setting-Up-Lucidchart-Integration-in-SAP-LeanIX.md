##  Setting Up Lucidchart Integration in SAP LeanIX
To set up the Lucidchart Integration, follow these steps:
  1. Navigate to the Diagrams tab.
  2. Click + New Diagram.
  3. Click Start setup now on the Lucidchart Integration overlay.
