##  Step 2: Create a SAML Application
  1. In SAP Cloud Identity Services, navigate to Applications & Resources > Applications.
  2. Choose Create to add a new application.
![Screenshot showing the Create button in the Applications list in SAP Cloud Identity Services.](https://help.sap.com/doc/72d375467c1e4dcb872dfa2998b6328d/CLOUD/en-US/loio21e2825c8bdb4627b9a973db234cc6ad_LowRes.png)
Create Application
  3. Complete the following fields:
     * Type: SAP LeanIX
     * Protocol Type: SAML 2.0
![Screenshot showing the application creation form with Type set to SAP LeanIX and Protocol Type set to SAML 2.0.](https://help.sap.com/doc/72d375467c1e4dcb872dfa2998b6328d/CLOUD/en-US/loiocf76cdd805fa4ec595248c92f865f73b_LowRes.png)
Application Configuration Fields
  4. Choose Create.
The application is added to the list of applications.
![Screenshot showing the newly created SAP LeanIX application in the applications list.](https://help.sap.com/doc/72d375467c1e4dcb872dfa2998b6328d/CLOUD/en-US/loio85ad6f788fbb4737bfbd31d0a8fafff0_LowRes.png)
Application Added to List
