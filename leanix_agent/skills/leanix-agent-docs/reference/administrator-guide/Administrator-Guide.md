# Administrator Guide
Learn how to manage workspace settings and user access in SAP LeanIX.
As an admin user of SAP LeanIX, you have access to the administration area where you can configure workspace settings and manage user access for your organization.
To navigate to the administration area, in the user profile menu, select Administration. You can browse the administration area using the sidebar on the left or the search field. For an overview of workspace settings, see [Workspace Settings Overview](https://help.sap.com/docs/leanix/ea/workspace-settings-overview?locale=en-US&state=PRODUCTION&version=CLOUD "Overview of settings available in the workspace administration area.").
If you're new to SAP LeanIX, see [SAP LeanIX for Enterprise Architects](https://help.sap.com/docs/leanix/ea/for-enterprise-architects?locale=en-US&state=PRODUCTION&version=CLOUD "First steps in using SAP LeanIX as an enterprise architect.") to learn how to get started.
If you're familiar with SAP LeanIX, find the topics that are most relevant to you in the corresponding sections of this guide.
