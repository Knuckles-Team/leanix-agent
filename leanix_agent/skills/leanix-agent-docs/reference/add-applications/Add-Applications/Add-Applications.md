##  Add Applications
**Note**
The table below doesn't list all available import methods. It includes recommended import methods to start with.
Once you’ve defined your initial scope of applications, import them using one of the following methods.
Import Methods for Applications Method | Description | When Recommended | Detailed Information
---|---|---|---
SAP discovery | Automatically discover SAP systems and services and add them to your workspace. | Your organization uses SAP systems and services. | [SAP Discovery](https://help.sap.com/docs/leanix/ea/sap-discovery?locale=en-US&state=PRODUCTION&version=CLOUD "Identify and manage your organization's SAP systems, services, and custom-built extensions")
SaaS discovery | Automatically discover your SaaS applications and add them to your workspace. | Your organization manages Software as a Service (SaaS) applications using third-party systems like Single-Sign-on (SSO), Secure Access Simplified (SASE), or Cloud Access Security Broker (CASB) solutions. | [SaaS Discovery](https://help.sap.com/docs/leanix/ea/saas-discovery?locale=en-US&state=PRODUCTION&version=CLOUD "SaaS discovery identifies your organization's SaaS applications by integrating with SSO, SASE, and CASB solutions. It then updates and enriches fact sheets using the reference catalog.")
Inventory builder |  Add data from architectural documents (images) using the inventory builder. You can import multiple fact sheet types at once by specifying their type. Inventory builder is an optional feature. You can activate it in your workspace. | You have architectural data in image format (pictures or diagrams). | [Inventory Builder](https://help.sap.com/docs/leanix/ea/inventory-builder?locale=en-US&state=PRODUCTION&version=CLOUD "The inventory builder uses AI to analyze diagrams, images, and text-based files. It extracts fact sheets and relations to help you build your inventory faster. You can review, adjust, and create fact sheets from discovered data.")
Excel import | Import data using Excel spreadsheets. You can add and update fact sheets of multiple types at once. | You want to import data using Excel spreadsheets manually. | [Importing Fact Sheet Data Through Excel File](https://help.sap.com/docs/leanix/ea/importing-fact-sheet-data-through-excel-file?locale=en-US&state=PRODUCTION&version=CLOUD "SAP LeanIX’s import feature enables efficient bulk updates, including adding, updating, and archiving fact sheets. This guide provides formatting rules, error handling, and step-by-step guidance for importing data through an Excel file.")
