##  Viewing Execution Details
Select an entry in the execution log to view details in the right-side pane. This includes:
  * Execution Log ID: Unique identifier for troubleshooting.
  * Timestamp: When the code was executed.
  * Code Execution Status: Execution result (success or error).
  * Input: Data provided to the code (in JSON format).
  * Output: Data returned by the code.


Use this information to troubleshoot failures and confirm expected behavior. Copy log details if needed.
