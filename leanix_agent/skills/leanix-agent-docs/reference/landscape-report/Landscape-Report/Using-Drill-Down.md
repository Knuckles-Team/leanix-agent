##  Using Drill-Down
The drill-down option in the report settings lets you examine the dependencies and relations for a fact sheet in detail. For example, you might drill down into an application landscape report by IT component to see which IT components have an unaddressed end-of-life risk.
To drill down, follow these steps:
  1. Choose Settings.
  2. From the Drill-Down dropdown, select the fact sheet type or relation.
  3. Optional: Under the Labels section, use the Drill-Down dropdown to choose the display label for the drill-down items.
  4. Click Apply.
