##  Available Integrations
Integrations for discovering AI agents:
  * AI Agent API: Upload AI agents through the API as agent cards based on the Agent-to-Agent (A2A) protocol. This lets you ingest your agents directly into SAP LeanIX without relying on integrations with agent-building platforms. For details, see [Uploading AI Agents Through the API](https://help.sap.com/docs/leanix/ea/discovering-ai-agents-using-api?locale=en-US&state=PRODUCTION&version=CLOUD "Upload AI agent cards to SAP LeanIX using the AI Agent API. Preview uploaded AI agents in the AI discovery inbox and add relevant fact sheets to your workspace.").
  * Microsoft Entra ID: Discover AI agents registered in Microsoft Entra ID through Microsoft’s Entra Agent ID framework. For details, see [Discovering AI Agents from Microsoft Entra ID](https://help.sap.com/docs/leanix/ea/discovering-ai-agents-from-microsoft-entra-id?locale=en-US&state=PRODUCTION&version=CLOUD "Learn how to integrate SAP LeanIX with Microsoft Entra ID to automate the discovery of AI agents.").
  * Google Cloud Engine: Discover AI agents built and deployed in Google Cloud Engine. For details, see [Discovering AI Agents from Google Agent Engine](https://help.sap.com/docs/leanix/ea/discovering-ai-agents-from-google-agent-engine?locale=en-US&state=PRODUCTION&version=CLOUD "Learn how to integrate SAP LeanIX with Google Agent Engine to automate the discovery of AI agents.").
  * Amazon Bedrock Agents: Discover AI agents registered in Amazon Bedrock Agents. For details, see [Discovering AI Agents from Amazon Bedrock](https://help.sap.com/docs/leanix/ea/discovering-ai-agents-from-amazon-bedrock?locale=en-US&state=PRODUCTION&version=CLOUD "Integrate SAP LeanIX with Amazon Bedrock Agents and Amazon Bedrock AgentCore to automatically discover your AI agents.").
  * Amazon Bedrock AgentCore: Discover AI agents registered in Amazon Bedrock AgentCore. For details, see [Discovering AI Agents from Amazon Bedrock](https://help.sap.com/docs/leanix/ea/discovering-ai-agents-from-amazon-bedrock?locale=en-US&state=PRODUCTION&version=CLOUD "Integrate SAP LeanIX with Amazon Bedrock Agents and Amazon Bedrock AgentCore to automatically discover your AI agents.").
  * SAP: Discover SAP AI agents by integrating SAP LeanIX with SAP Cloud ALM. For details, see [Discovering SAP AI Agents](https://help.sap.com/docs/leanix/ea/discovering-sap-ai-agents?locale=en-US&state=PRODUCTION&version=CLOUD "Learn how to integrate SAP LeanIX with SAP Cloud ALM to automate the discovery of SAP AI agents.").


Integration for discovering MCP servers:
  * Microsoft Azure API Center: Connect to the Microsoft Azure API Center to discover, reuse, and manage the MCP servers registered within Azure API Management. For details, see [Discovering MCP Servers from Microsoft Azure API Center](https://help.sap.com/docs/leanix/ea/discovering-mcp-servers-from-microsoft-azure-api-center?locale=en-US&state=PRODUCTION&version=CLOUD "Discover your MCP servers registered in Microsoft Azure API Center. Review discovered items and add them to your inventory in SAP LeanIX.").
