##  Activating the Dashboard
If the onboarding dashboard is not yet activated in your workspace, admin users can activate it. The latest version of the dashboard becomes available to all workspace users. To learn more about managing dashboards, see [Dashboard Templates](https://help.sap.com/docs/leanix/ea/dashboards-settings?locale=en-US&state=PRODUCTION&version=CLOUD#loio2759a5e37a441014acd0dfe4da61c1c5__dashboard_templates).
Follow these steps:
  1. Go to the Dashboards section in the administration area.
  2. Locate the Templates settings group, click Add template for the Enterprise Architect Onboarding Dashboard template, then confirm your action.
